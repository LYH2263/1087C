Bookshop API Docs

示例接口文档，提供前后端联调参考。

认证

  • POST /api/auth/register 注册
  • POST /api/auth/login 登录
  • POST /api/auth/logout 退出
  • POST /api/auth/refresh 刷新 Token
  • POST /api/auth/forgot-password 忘记密码
  • POST /api/auth/reset-password 重置密码

书籍与订单

  • GET /api/books 书籍查询
  • POST /api/cart 加入购物车
  • POST /api/orders/checkout 生成订单
